📋 Apps Script Setup (one time only)
  1. Open your Google Sheet → click Extensions → Apps Script
  2. Delete existing code → paste the full EyeMedia_FMS_AppsScript.gs file
  3. Click Deploy → New deployment → Web app
  4. Execute as: Me · Who has access: Anyone (NOT "Anyone with Google account")
  5. Click Deploy → copy Web App URL → paste above → click Connect
📌 Make it work on phones & all devices automatically:
After connecting, open the index.html file in a text editor → find the line:
const HARDCODED_SHEETS_URL = '';
Paste your URL between the quotes → save → re-upload to GitHub. Every device will then connect automatically with no manual setup.
